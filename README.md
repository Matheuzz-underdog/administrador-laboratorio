# Sistema de Gestión de datos de Laboratorio
###### administrador-laboratorio
---
Primeros pasos para la creación de un administrador de datos implementando Node.Js y ExpressJS. 
### Instrucciones
En esta primera fase, el sistema carece de una interfaz gráfica, por ende, las consultas deben hacerse con la ayuda de Thunder Client (o utilidades del mismo tipo).
